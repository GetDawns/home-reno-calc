## Naming convention

- URL slug = lowercase keyword phrase joined by `-`, ending in `-calculator` for the page itself
- File names mirror the slug
- Math function name: `calculate<Slug>`, e.g. `calculatePaint`, `calculateTile`
- React component: `<Slug>Calculator.tsx`, e.g. `PaintCalculator.tsx`
- Default export object: `<SLUG>_DEFAULTS`, e.g. `PAINT_DEFAULTS`

## How to extend

1. Pick a row from this table. Verify volume/KD with current Ahrefs/Semrush before committing.
2. Build math (`lib/calculators/<slug>.ts`) — TDD recommended. Hand-check 3 example inputs against an authoritative source.
3. Build UI (`components/calculators/<Slug>Calculator.tsx`) — copy paint or mulch as template.
4. Register in `components/calculators/registry.ts`.
5. Add manifest entry in `content/calculators/manifest.ts` (slug, title, FAQs, shop affiliates, related, etc.).
6. Write 1–3 supporting blog articles (e.g., "How much grout do I need for a 100 sq ft tile floor?") that internally link to the calculator.
